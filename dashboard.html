☁️ Synchronisation cloud
Non configuré
Connectez une base de données Supabase gratuite pour sauvegarder vos données dans le cloud. Vos données survivront à tout changement de navigateur ou d'appareil.
Comment configurer (2 minutes) :
  1. Allez sur supabase.com → Créer un compte gratuit
  2. Créez un nouveau projet (choisissez une région Europe)
  3. Dans Settings → API → copiez l'URL et la clé anon public
  4. Dans Table Editor → cliquez "New Table" → nommez-la patrimonia_data
  5. Ajoutez ces colonnes : username (text, primary key) · data (jsonb) · updated_at (timestamptz)
  6. Collez l'URL et la clé ci-dessous et cliquez Connecter

💡 Optimisations fiscales personnalisées
📗 PEA
Avantage : Plus-values et dividendes exonérés d'IR après 5 ans (seulement 17.2% prélèvements sociaux)
Plafond : 150 000 € de versements
Idéal pour : ETF monde, actions européennes
📘 Assurance-Vie
Avantage : Abattement de 4 600€/an (9 200€ couple) sur les gains après 8 ans
Taux : 7.5% IR + 17.2% PS (si < 150k€ versé)
Idéal pour : Diversification, transmission
📙 PER
Avantage : Versements déductibles du revenu imposable (dans la limite de ~10% salaire brut)
Plafond : 10% des revenus N-1 (max ~35 194€)
Idéal pour : TMI élevée, préparer la retraite
